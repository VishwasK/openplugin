# Web Search Command

Searches the web for information and retrieves relevant results.

## Usage

This command takes a search query and returns relevant web search results that can be used to answer questions or gather information.

## Input Parameters

- **query**: The search query or question to search for
- **num_results**: Number of results to return (default: 5)
- **region**: Optional region/language for search results

## Output

Returns search results including:
- Title of each result
- URL/link to the source
- Snippet/preview of the content
- Relevance to the query

## Behavior

- Performs web search using search engine
- Retrieves top relevant results
- Extracts key information from results
- Formats results in a useful way
- Can be used to answer questions with current information

## Examples

- "Search for latest news about AI developments"
- "Find information about Python async programming"
- "What are the current trends in machine learning?"

## Use Cases

- Answer questions with current information
- Research topics
- Find recent news or updates
- Get factual information from the web
- Verify information
