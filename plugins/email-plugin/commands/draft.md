# Draft Email Command

Drafts a professional email based on user requirements.

## Usage

This command takes user input describing what email they want to send and generates a well-structured email draft.

## Input Parameters

- **recipient**: Who the email is for
- **subject**: What the email is about
- **purpose**: What you want to achieve with the email
- **tone**: Desired tone (professional, casual, formal, friendly, etc.)
- **key_points**: Important points to include

## Output

Returns a complete email draft with:
- Subject line
- Greeting
- Body paragraphs
- Closing
- Signature suggestion

## Behavior

- Generates contextually appropriate content
- Adapts tone based on requirements
- Includes all key points mentioned
- Ensures professional formatting
- Suggests appropriate closing
