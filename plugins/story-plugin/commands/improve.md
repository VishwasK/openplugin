# Improve Story Command

Improves and refines an existing story.

## Usage

This command takes a story and enhances it with better writing, descriptions, and flow.

## Input Parameters

- **story**: The story text to improve
- **focus**: What to focus on (dialogue, descriptions, pacing, character development, etc.)
- **style**: Desired writing style (literary, casual, poetic, etc.)

## Output

Returns an improved version that:
- Has better flow and pacing
- Includes richer descriptions
- Has more natural dialogue
- Shows rather than tells
- Maintains the original plot and characters

## Behavior

- Enhances writing quality while keeping the core story
- Adds descriptive details where appropriate
- Improves dialogue to be more natural
- Maintains consistency with original
- Can focus on specific aspects if requested

## Examples

- "Improve this story with better descriptions"
- "Enhance the dialogue in this story"
- "Make this story more literary in style"
