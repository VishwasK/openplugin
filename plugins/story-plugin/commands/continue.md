# Continue Story Command

Continues an existing story from where it left off.

## Usage

This command takes an existing story and continues it, maintaining consistency with the original.

## Input Parameters

- **story**: The existing story text to continue
- **direction**: Optional direction for how to continue (e.g., "add a twist", "introduce a new character", "build suspense")
- **length**: How much to add (short continuation, full chapter, etc.)

## Output

Returns a continuation that:
- Maintains the same style and tone
- Keeps character consistency
- Follows the established plot
- Adds new developments naturally

## Behavior

- Analyzes the existing story for style, tone, and plot
- Continues seamlessly from where it left off
- Maintains character voices and personalities
- Respects the established world and rules
- Can add new elements if requested

## Examples

- "Continue this story: [story text]"
- "Continue the story but add a plot twist"
- "Add another chapter to this story"
