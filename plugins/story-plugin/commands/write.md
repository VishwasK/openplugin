# Write Story Command

Generates a creative story based on user prompts and preferences.

## Usage

This command takes user input describing what kind of story they want and generates a complete, engaging story.

## Input Parameters

- **genre**: Type of story (fantasy, sci-fi, mystery, romance, adventure, etc.)
- **theme**: Main theme or topic
- **characters**: Character descriptions or names
- **setting**: Where the story takes place
- **length**: Desired length (short, medium, long)
- **tone**: Story tone (funny, serious, dark, lighthearted, etc.)

## Output

Returns a complete story with:
- Engaging opening
- Well-developed plot
- Character development
- Satisfying conclusion
- Appropriate length for the request

## Behavior

- Creates original, creative content
- Maintains consistent tone throughout
- Develops characters and plot naturally
- Adapts length based on user preference
- Ensures the story has a beginning, middle, and end

## Examples

- "Write a fantasy story about a young wizard discovering their powers"
- "Create a short sci-fi story set on Mars with a detective character"
- "Write a funny adventure story about a talking cat"
