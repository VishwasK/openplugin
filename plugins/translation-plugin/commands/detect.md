# Detect Language Command

Detects the language of given text.

## Usage

This command identifies what language text is written in.

## Input Parameters

- **text**: Text to analyze

## Output

Returns:
- Detected language name
- Confidence level
- Alternative languages if ambiguous

## Behavior

- Identifies language accurately
- Provides confidence level
- Handles mixed-language text
- Works with short and long texts

## Examples

- "What language is this: Bonjour, comment allez-vous?"
- "Detect the language of this text"
