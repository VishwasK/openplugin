# Translate Command

Translates text from one language to another.

## Usage

This command takes text and translates it to a target language while preserving meaning and context.

## Input Parameters

- **text**: Text to translate
- **target_language**: Target language (e.g., Spanish, French, German)
- **source_language**: Source language (optional, auto-detect if not provided)
- **context**: Optional context to improve translation quality

## Output

Returns translated text that:
- Accurately conveys the original meaning
- Uses natural phrasing in target language
- Preserves tone and style
- Maintains technical terms appropriately

## Behavior

- Translates text accurately
- Preserves meaning and context
- Uses natural language in target language
- Handles technical terms appropriately
- Can auto-detect source language

## Examples

- "Translate this to Spanish: Hello, how are you?"
- "Translate this technical document to French"
- "Translate this code comment to Japanese"
