# Translation Plugin

A plugin for translating text between languages using LLM.

## Features

- **Translate**: Translate text to any language
- **Detect Language**: Identify the language of text
- **Context-Aware**: Preserves meaning and context

## Commands

### `/translate`
Translate text to target language.

Example:
```
Translate this to Spanish: Hello, how are you?
```

### `/detect`
Detect the language of text.

Example:
```
What language is this: Bonjour!
```

## Usage

See `examples/translation_app.py` for a complete example.

## Use Cases

- Multi-language content
- Localization
- Language learning
- Content translation
- International communication
