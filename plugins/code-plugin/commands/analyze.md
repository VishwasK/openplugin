# Analyze Code Command

Analyzes code quality, identifies issues, and suggests improvements.

## Usage

This command takes code and provides comprehensive analysis including code quality, potential bugs, performance issues, and best practices.

## Input Parameters

- **code**: The code to analyze
- **language**: Programming language (Python, JavaScript, etc.)
- **focus**: What to focus on (performance, security, style, bugs, etc.)

## Output

Returns analysis including:
- Code quality assessment
- Potential bugs or issues
- Performance concerns
- Best practice violations
- Specific improvement suggestions

## Behavior

- Analyzes code structure and logic
- Identifies common patterns and anti-patterns
- Suggests specific improvements
- Provides code examples for improvements
- Considers language-specific best practices

## Examples

- "Analyze this Python code for performance issues"
- "Check this JavaScript code for security vulnerabilities"
- "Review this code for best practices"
