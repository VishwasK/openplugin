# Refactor Code Command

Suggests refactoring improvements for code.

## Usage

This command takes code and suggests how to refactor it for better maintainability, readability, or performance.

## Input Parameters

- **code**: The code to refactor
- **language**: Programming language
- **goals**: Refactoring goals (readability, performance, maintainability, etc.)

## Output

Returns refactoring suggestions including:
- Refactored code examples
- Explanation of changes
- Benefits of refactoring
- Before/after comparisons

## Behavior

- Suggests specific refactoring patterns
- Provides improved code examples
- Explains why changes improve the code
- Maintains functionality while improving structure
- Follows language-specific best practices

## Examples

- "Refactor this code for better readability"
- "Improve this code's performance"
- "Make this code more maintainable"
