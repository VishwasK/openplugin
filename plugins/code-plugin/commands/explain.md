# Explain Code Command

Explains what code does in plain language.

## Usage

This command takes code and explains what it does, how it works, and what each part does.

## Input Parameters

- **code**: The code to explain
- **language**: Programming language
- **detail_level**: How detailed the explanation should be (simple, detailed, comprehensive)

## Output

Returns explanation including:
- Overall purpose of the code
- Step-by-step breakdown
- Explanation of key concepts
- What each function/class does
- How data flows through the code

## Behavior

- Explains code in plain language
- Breaks down complex logic
- Identifies key concepts and patterns
- Provides context for understanding
- Can explain at different detail levels

## Examples

- "Explain this Python function"
- "What does this code do?"
- "Explain this algorithm step by step"
