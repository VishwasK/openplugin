# Code Plugin

A plugin for analyzing, explaining, and improving code using LLM.

## Features

- **Code Analysis**: Identify issues and suggest improvements
- **Code Explanation**: Explain code in plain language
- **Code Refactoring**: Suggest refactoring improvements

## Commands

### `/analyze`
Analyze code quality and identify issues.

Example:
```
Analyze this Python code for performance issues:
[code here]
```

### `/explain`
Explain what code does.

Example:
```
Explain this function:
[code here]
```

### `/refactor`
Suggest refactoring improvements.

Example:
```
Refactor this code for better readability:
[code here]
```

## Usage

See `examples/code_app.py` for a complete example.

## Use Cases

- Code review assistance
- Learning from code
- Code improvement
- Debugging help
- Understanding legacy code
