# Text Processing Plugin

A plugin for processing, summarizing, and extracting information from text.

## Features

- **Summarize**: Create concise summaries
- **Extract**: Extract specific information
- **Process**: Various text processing tasks

## Commands

### `/summarize`
Summarize long text.

Example:
```
Summarize this article in 3 paragraphs:
[article text]
```

### `/extract`
Extract specific information.

Example:
```
Extract all dates and names from this text:
[text]
```

## Usage

See `examples/text_processing_app.py` for a complete example.

## Use Cases

- Document summarization
- Information extraction
- Content analysis
- Data mining from text
- Quick content review
