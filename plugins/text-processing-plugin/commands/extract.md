# Extract Information Command

Extracts specific information from text.

## Usage

This command extracts specific types of information like dates, names, facts, or key points from text.

## Input Parameters

- **text**: Text to extract from
- **what**: What to extract (dates, names, facts, key points, numbers, etc.)
- **format**: Output format (list, structured, json, etc.)

## Output

Returns extracted information in requested format:
- Structured data
- Lists of items
- Key-value pairs
- Formatted output

## Behavior

- Identifies relevant information
- Extracts accurately
- Formats output appropriately
- Handles various text types
- Can extract multiple types at once

## Examples

- "Extract all dates from this text"
- "Extract key facts from this article"
- "Extract names and locations from this document"
