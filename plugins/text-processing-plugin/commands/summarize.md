# Summarize Text Command

Summarizes long text into shorter, concise versions.

## Usage

This command takes long text and creates a summary that captures the key points.

## Input Parameters

- **text**: Text to summarize
- **length**: Desired summary length (short, medium, long, or word count)
- **focus**: What to focus on (main points, key facts, conclusions, etc.)

## Output

Returns a summary that:
- Captures key points
- Maintains important information
- Is concise and readable
- Preserves main ideas

## Behavior

- Extracts key information
- Maintains important details
- Creates coherent summary
- Adapts length to requirements
- Can focus on specific aspects

## Examples

- "Summarize this article in 3 sentences"
- "Create a bullet-point summary of this text"
- "Summarize focusing on the main conclusions"
